Lab3: Спортивний Інвентар

Опис

Даний проєкт виконує операції з набором спортивного інвентарю, зокрема:

1. Відображення загальної інформації про весь інвентар.
2. Сортування інвентарю за зростанням ціни (виводяться тільки назва та ціна).
3. Сортування інвентарю за спаданням кількості (виводяться тільки назва та кількість).

Результати роботи програми відображаються у консолі.

Файли

1. Lab3Task.java
   Містить основну логіку:
   - Метод `main` для ініціалізації списку інвентарю та виконання операцій сортування.
   - Вивід результатів у консоль у зручному форматі.

Системні вимоги

- Java Development Kit (JDK) версії 8 або новішої.
- Текстовий редактор або середовище розробки (IDE), наприклад IntelliJ IDEA, Eclipse, або VS Code.

Інструкція з використання

1. Завантажте файл
   Завантажте файл `Lab3Task.java` у вашу локальну файлову систему.

2. Компілюйте проєкт
   Відкрийте термінал або командний рядок у директорії, де знаходиться файл, і виконайте команду:
   javac lab3/Lab3Task.java

3. Запустіть програму
   Після успішної компіляції запустіть клас `Lab3Task` за допомогою команди:
   java lab3.Lab3Task

4. Результати роботи
   У консолі ви побачите:
   - Загальну інформацію про інвентар.
   - Список інвентарю, відсортований за зростанням ціни (назва та ціна).
   - Список інвентарю, відсортований за спаданням кількості (назва та кількість).

Як змінити вхідні дані

1. Відкрийте файл `Lab3Task.java` у текстовому редакторі або IDE.
2. Змініть список інвентарю в методі `main`:
   inventoryList.add(new Lab3Task("Новий м'яч", 40.0, 0.5, "Puma", "М'яч", 25));
3. Збережіть файл і повторіть кроки компіляції та запуску.

Додаткова інформація

- Програма гарантує коректне сортування та вивід інформації за заданими критеріями.
