Lab5: Наслідування та поліморфізм

Опис

Цей проєкт виконує такі операції з солодощами:

1. Додавання цукерок різних типів до подарунка.
2. Обчислення:
   - Загальної ваги подарунка.
   - Сортування цукерок за вагою.
   - Пошуку шоколадних цукерок у заданому діапазоні вмісту шоколаду.

Результати роботи відображаються у консолі.

Файли

1. Lab5Task.java  
   Головний клас програми:
   - Створює об'єкти солодощів різних типів.
   - Додає їх до подарунку та демонструє функціонал програми.

2. Sweet.java
   Базовий клас для всіх солодощів:
   - Містить загальні властивості: назва, вага.

3. Candy.java
   Клас для цукерок:
   - Додає поле вмісту цукру.

4. ChocolateCandy.java  
   Клас для шоколадних цукерок:
   - Додає поле вмісту шоколаду.

5. Lollipop.java  
   Клас для льодяників:
   - Додає поле смаку.

6. GummyBear.java  
   Клас для жувальних ведмедиків:
   - Додає поле кольору.

7. Gift.java  
   Клас для подарунку:
   - Містить список об'єктів солодощів.
   - Забезпечує операції додавання, сортування та пошуку.

Системні вимоги

- Java Development Kit (JDK) версії 8 або новішої.
- Текстовий редактор або середовище розробки (IDE), наприклад IntelliJ IDEA, Eclipse або VS Code.

Інструкція з використання

1. Завантаження файлів
   Завантажте файли проєкту в одну директорію.

2. Компіляція проєкту
   Відкрийте термінал або командний рядок у директорії з файлами та виконайте команду:
   javac *.java

3. Запуск програми
   Запустіть програму, виконавши команду:
   java Lab5Task

4. Результати роботи
   У консолі ви побачите:
   - Загальну вагу подарунку.
   - Відсортовані за вагою цукерки.
   - Список шоколадних цукерок у заданому діапазоні вмісту шоколаду.

Приклад використання

1. Ініціалізація солодощів у Lab5Task.java:
   java
   ChocolateCandy candy1 = new ChocolateCandy("КітКат", 50, 60, 40);
   Lollipop candy2 = new Lollipop("Чупа Чупс", 20, 80, "полуничний");
   GummyBear candy3 = new GummyBear("Зелені ведмедики Харібо", 30, 70, "зелений");
   

2. Результат виконання:
   Вага подарунку: 100г
   Солодощі відсортовані за вагою:
   Чупа Чупс (20г, вміст цукру 80%, полуничний смак)
   Зелені ведмедики Харібо (30г, вміст цукру 70%, зелений колір)
   КітКат (50г, вміст цукру 60%, вміст шоколаду 40%)
   Солодощі зі вмістом шоколаду від 30% до 50%:
   КітКат (50г, вміст цукру 60%, вміст шоколаду 40%)

Як змінити вхідні дані

1. Відкрийте файл Lab5Task.java у текстовому редакторі.
2. Додайте або змініть об'єкти солодощів, використовуючи відповідні класи.
3. Збережіть файл і повторіть кроки компіляції та запуску.

Додаткова інформація

- Якщо вхідні дані некоректні (наприклад, негативна вага), програма повідомить про помилку.
