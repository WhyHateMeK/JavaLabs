Lab4Task: Відношення між класами в мові програмування Java

Опис
Цей проект реалізує обробку тексту з використанням спеціально створених класів:
1. Letter — представляє окрему літеру тексту.
2. Word — складається з масиву літер.
3. Sentence — складається з масиву слів та розділових знаків.
4. Text — складається з масиву речень.

Функціональність:
1. Видаляє всі попередні входження останньої літери в кожному слові (незалежно від регістру).
2. Замінює всі послідовності пробілів і табуляцій одним пробілом.
3. Виводить оброблений текст у консоль.

Системні вимоги
- Java Development Kit (JDK) версії 8 або новішої.
- Інструмент для компіляції та запуску Java-коду (наприклад, IntelliJ IDEA, Eclipse або командний рядок).

Інструкція з використання

1. Завантаження файлів

Завантажте файл Lab4Task.java і помістіть його в локальну файлову систему. Код організований у пакеті `lab4`, тому його структура має виглядати так:
lab4/
    Lab4Task.java
2. Компіляція

Перейдіть у терміналі до директорії, де знаходиться папка `lab2`, і виконайте команду компіляції:
javac lab4/Lab4Task.java

3. Запуск

Запустіть програму, виконавши команду:
java lab4.Lab4Task

4. Використання програми

1. Після запуску програма попросить ввести текст. Введіть текст у консоль і натисніть Enter.
2. Програма виконає наступні дії:
   - Видалить усі попередні входження останньої літери в кожному слові.
   - Замінить усі послідовності пробілів і табуляцій на один пробіл.
3. Результат обробки буде виведено в консоль.

Приклад використання
Вхідний текст:
    Я                люблю         Україну
Вихідний результат:
Я лблю країну
