Lab2Task

Lab2Task - це програма на Java, яка обробляє введений текст. У кожному слові тексту видаляються всі попередні входження останньої літери цього слова, незалежно від регістру (великі чи маленькі літери). Програма використовує лише тип даних `String` для роботи з рядками, як зазначено в завданні.

Вимоги
- Java Development Kit (JDK) версії 8 або вище.
- Інструмент для компіляції та запуску Java-коду (наприклад, командний рядок або IDE).

Інструкція по запуску

1. Завантаження та підготовка файлів
1. Завантажте файл `Lab2Task.java` з репозиторію.
2. Збережіть файл у відповідній директорії на вашому комп'ютері.

2. Компіляція програми
- Відкрийте термінал або командний рядок.
- Перейдіть у директорію, де збережено файл `Lab2Task.java`.
- Виконайте команду:
  javac Lab2Task.java
  Це створить файл `Lab2Task.class` у цій самій директорії.

3. Запуск програми
- У тому ж терміналі виконайте команду:
  java Lab2Task
- Програма запуститься і виведе запит на введення тексту.

### 4. Використання програми
1. Введіть текст у консоль, наприклад:
   Введіть текст: Я люблю Україну
2. Програма обробить введений текст і виведе результат. Наприклад:
   Результат: Я лблю країну

5. Закриття програми
- Програма завершується автоматично після виведення результату.

Особливості
- Програма враховує регістр (великі та маленькі літери): однакові літери різного регістру вважаються однаковими.
- Оригінальний регістр тексту зберігається у результаті.
- Для всіх операцій із рядками використовується лише тип `String`.

Обробка помилок
Програма може вивести повідомлення про помилки у разі виникнення винятків, наприклад, якщо введення не відповідає очікуваному формату. У такому випадку буде виведено повідомлення:
Виникла помилка: [Опис помилки]
