Lab6: Робота з колекціями в мові програмування Java

Опис

Цей проєкт реалізує типізовану колекцію Lab6Task, що базується на однозв'язному списку та реалізує інтерфейс `Set`. Клас підтримує операції для роботи з унікальними елементами, включаючи додавання, видалення, перевірку наявності та інші стандартні методи `Set`.

Особливості:

1. Реалізація однозв'язного списку для внутрішньої структури даних.
2. Підтримка трьох типів конструкторів:
   - Порожній конструктор.
   - Конструктор із одним елементом.
   - Конструктор, який приймає стандартну колекцію.
3. Метод `toString` для зручного відображення вмісту колекції.
4. Повна відповідність інтерфейсу `Set`.

Файли

1. Lab6Task.java  
   Реалізує основну функціональність типізованої колекції:
   - Методи для додавання, видалення, перевірки наявності елементів.
   - Реалізація ітератора для проходження елементів у циклі `for-each`.
   - Конструктори для різних способів ініціалізації.

2. Lab6Main.java  
   Демонструє використання класу Lab6Task:
   - Створює кілька колекцій із різними конструкторами.
   - Виконує основні операції над колекцією.
   - Виводить результати роботи.

Системні вимоги

- Java Development Kit (JDK) версії 8 або новішої.
- Текстовий редактор або середовище розробки (IDE), наприклад IntelliJ IDEA, Eclipse або VS Code.

Інструкція з використання

1. Завантажте файли  
   Завантажте файли Lab6Task.java і Lab6Main.java у вашу локальну файлову систему.

2. Компілюйте проєкт  
   Відкрийте термінал або командний рядок у директорії, де знаходяться файли, і виконайте команду:
   javac lab6/Lab6Task.java lab6/Lab6Main.java

3. Запустіть програму  
   Після успішної компіляції запустіть клас Lab6Main за допомогою команди:
   java lab6.Lab6Main

4. Результати роботи  
   У консолі ви побачите:
   - Стан колекцій після різних операцій (додавання, видалення тощо).
   - Результати ітерації по елементах колекції.
   - Демонстрацію роботи методів конструктора.

Як змінити вхідні дані

1. Відкрийте файл `Lab6Main.java` у текстовому редакторі або IDE.
2. У методі main змініть значення, які передаються до колекцій:
   java
   Lab6Task<String> collection = new Lab6Task<>();
   collection.add("Яблуко");
   collection.add("Банан");
   collection.add("Вишня");
3. Збережіть файл і повторіть кроки компіляції та запуску.

Додаткова інформація

- Клас Lab6Task забезпечує уникнення дублювання елементів. Якщо елемент вже існує в колекції, він не додається.
- Клас підтримує перевірку відповідності інтерфейсу `Set`.
